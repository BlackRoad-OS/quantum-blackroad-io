Quantum Simulator

Bell State Example

Create a maximally entangled Bell state using Hadamard and CNOT gates.

from blackroad.quantum import QuantumCircuit, Simulator # Create circuit qc = QuantumCircuit(2) # Create Bell state qc.h(0) # Hadamard on qubit 0 qc.cx(0, 1) # CNOT from qubit 0 to 1 qc.measure_all() # Run simulation sim = Simulator() result = sim.run(qc, shots=1000) print(result.counts())
Measurement Results (1000 shots):
{'00': 501, '11': 499}

‚úì Perfect entanglement! Qubits are always measured in the same state (both 0 or both 1).

Quick Start Guide

# Install BlackRoad Quantum SDK pip install blackroad-quantum # Import libraries from blackroad.quantum import QuantumCircuit, Simulator # Create your first quantum circuit qc = QuantumCircuit(2) # Apply quantum gates qc.h(0) # Hadamard gate qc.cx(0, 1) # CNOT gate qc.measure_all() # Measure all qubits # Run on simulator sim = Simulator() result = sim.run(qc, shots=1000) # View results print(result.counts()) # Output: {'00': 501, '11': 499}
